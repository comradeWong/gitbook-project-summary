# Vue Cli 3 项目搭建总结

### 搭建步骤

> 得益于Vue Cli 3 ，Vue 有了自己的图形管理界面，使用vue ui即可进入，在这儿不做介绍，只介绍一下控制台新建项目的步骤。

#### **1. 首先你总得安装这个Vue Cli 3 的包吧**。（我的版本为3.9.3）

```text
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

**2. 然后运行以下指令来创建一个新项目。**

```text
vue create <项目名称>
```
